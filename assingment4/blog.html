Question 3

What is Event Bubble?

Event Bubble is an event which bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.

What is callback function and what it does?

Callback function is a function which passed into another function as an argument, which is inside a outer function to complete some kind of routine or action.
